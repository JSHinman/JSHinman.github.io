Running Motor on Button Press

The first thing you have to do is add the controller. Go into the robot configuration menu and select Add Device and then controller and wihtout changing anything, press done. Inside the int main() function, you need a while loop set to always active. To see if the button is being pressed, you need an if statement. Inside that you detect if a particular button is being pressed and if so run the motor. To also make the motor spin in reverse, put an else if after it and have it check if another button is being pressed. To make it stop after we let go of the buttons, is to have an else statement after the if statements. Put a NameMotor.stop() inside the else. This means that if neither of the buttons are being pressed, the motor will stop spinning.

1. while(true) {
2.    if (Controller1.ButtonLeft.pressing()) {
3.       NameOfMotor.spin(forward, 100, vex::velocityUnits::pct);
4.    } if (Controller1.ButtonRight.pressing()) {
5.       NameOfMotor.spin(reverse, 100, vex::velocityUnits::pct);
6.    } else {
7.       NameOfMotor.stop();
8. }
